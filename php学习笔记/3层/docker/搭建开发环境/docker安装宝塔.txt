docker pull pch18/baota:clear
pch18/baota或pch18/baota:latest等同pch18/baota:lnmp
pch18/baota:lnmp为最新版本的官方纯净安装的基础上安装nginx,mysql,php
pch18/baota:lnp 为官方版本纯净安装的基础上安装nginx,php(不内置mysql,用于外置数据库的环境)
pch18/baota:lamp 为官方版本纯净安装的基础上安装apache,php
pch18/baota:lap 为官方版本纯净安装的基础上安装apache,php(不内置mysql,用于外置数据库的环境)
pch18/baota:clear 为官方版本纯净安装, 不默认安装nginx,mysql,php等程序

拉取镜像好以后，先创建一个volume数据卷。

docker volume create baota

启动容器
docker run -tid --name baota -p 80:80 -p 443:443 -p 3306:3306 -p 8888:8888 -p 888:888 -p 8022:8022 --privileged=true --shm-size=1g --restart always -v baota:/www -v ~/wwwroot:/www/wwwroot pch18/baota:clear

/www文件夹建议保存在volume卷中, /www/wwwroot建议映射到宿主机的目录下,方便上传网站代码等文件，修改第二个-v参数。


docker exec -it baota /bin/bash

输入命令"bt"修改宝塔登录账号和密码。

选择第5项和第6项进行修改，修改完成后通过http://ip:8888访问。到这里基本就算完成了。剩下的就是宝塔的使用了。
